# Portfolio-CLI-Mwinza

**How to run Portfolio-CLI-Mwinza**
1. Install Node.js which is the required environment.  
On windows, visit [node website](https://nodejs.org) and follow the instructions.  
On MacOS, type `brew install node` the terminal.  
On Linux, look up a guide for your respective distribution.  
  
2. Run it by typing `node <filepath>` in your terminal.
